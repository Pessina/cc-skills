# Daml CLI (DPM)

Reference for the DPM (Digital Asset Package Manager) CLI, the primary tool for Daml SDK 3.x projects.

## Structure

- `SKILL.md` - Skill metadata and trigger description
- `AGENTS.md` - Complete compiled reference for AI agents
- `metadata.json` - Version and reference metadata

## Topics Covered

- Installation and Java requirements
- `daml.yaml` configuration (full field reference)
- `dpm build` / `dpm test` / `dpm sandbox` / `dpm codegen-js` / `dpm clean`
- Project setup and development workflows
- Troubleshooting common issues
- Project structure conventions

## Installation

```bash
cp -r daml-cli/ ~/.claude/skills/daml-cli/
```
