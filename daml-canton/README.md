# Canton Operations

Reference for Canton blockchain node setup and operations with Daml 3.x.

## Structure

- `SKILL.md` - Skill metadata and trigger description
- `AGENTS.md` - Complete compiled reference for AI agents
- `metadata.json` - Version and reference metadata

## Topics Covered

- Starting Canton sandbox with DPM
- Party allocation and user creation via REST API
- DAR deployment and upgrade rules
- JSON Ledger API v2 endpoints and command submission
- Active contract queries
- Canton topology (domain, participant, sequencer, mediator)
- Health checks and logging
- Troubleshooting common issues
- Full development workflow

## Installation

```bash
cp -r daml-canton/ ~/.claude/skills/daml-canton/
```
